# Inform 7 Interactive Fiction Game

## Overview
This project is a text based interactive fiction game built using Inform 7. The game focuses on narrative driven adventure with unexpected plots and twists.

The player progresses through the story by typing commands to explore environments and interact with objects, where each decision can affect the outcome of the game.

---

## Playable Build
A compiled playable version of the game is included in this repository:

- File: Fictiongame.gblorb`

This is a Glulx format interactive fiction file.

---

## How to Play

To run the game, you need an interactive fiction interpreter:

Recommended interpreters:
- Lectrote 
- Gargoyle 

Steps:
1. Download Fictiongame.gblorb`
2. Open it using Lectrote or Gargoyle
3. Start playing using text commands

Example commands:
- take it
- go to front yard
- stay
- take risky surgery
- ignore kelly

---

## Technologies Used
- Inform 7 
- Glulx virtual machine format

---

## Development Instructions

To build the game from source:

1. Open the project in Inform 7
2. Navigate to the "Go" menu
3. Select "Go → Release for Glulx"
4. The compiled `.gblorb` file will be generated in the project build output folder
5. Move the file into the `builds/` directory for distribution
